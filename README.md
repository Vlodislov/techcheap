# TechCheap — Backend для Блог-Платформы

Современное, высокопроизводительное RESTful API для блог-платформы, построенное на базе фреймворка FastAPI. Проект реализует модульную архитектуру, масштабируемую систему разграничения прав доступа и продвинутые механизмы кеширования.

---

## 🚀 Ключевые достижения и оптимизация

* **Оптимизация производительности (Latency):** За счет внедрения агрессивного кеширования на уровне Redis и оптимизации тяжелых SQL-запросов (исключение проблемы `N+1`), среднее время ответа API (TTFB) было снижено **с 800 ms до 120 ms** под нагрузкой.
* **Полнотекстовый поиск:** Реализована эффективная система тегов и полнотекстового поиска по статьям, обеспечивающая релевантную и быструю выдачу контента.
* **Отказоустойчивость:** Настроена автоматизированная система создания резервных копий базы данных (Backups) по расписанию, гарантирующая сохранность данных.

---

## 🛠 Функционал и возможности

* **Модульная архитектура:** Четкое разделение кодовой базы на независимые слои (роутеры, бизнес-логика, доступ к данным, конфигурация), что упрощает масштабирование и поддержку.
* **RESTful API для контента:** Полный цикл CRUD-операций для публикации статей, управления категориями и интерактивной системы комментариев.
* **Многоуровневая система доступа (RBAC):** Реализовано разграничение прав пользователей (гость, авторизованный пользователь, модератор, администратор).
* **Безопасность:** Надежная аутентификация и авторизация на базе JWT-токенов.
* **Сквозное логирование и обработка ошибок:** Изолированный централизованный модуль логирования и глобальный перехват исключений (Middlewares/Exception Handlers) для предсказуемого поведения системы.

---

## 🧰 Стек технологий

* **Backend:** Python 3.11+, FastAPI, Pydantic (валидация данных).
* **Database & ORM:** PostgreSQL, SQLAlchemy / Tortoise ORM, Alembic (миграции).
* **Caching & Broker:** Redis.
* **DevOps & Infrastructure:** Docker, Docker Compose, CI/CD.

---

## 📦 Архитектура проекта

Кодовая база спроектирована с учетом принципа разделения ответственности (Separation of Concerns):

```text
techcheap/
│
├── app/
│   ├── api/            # Эндпоинты (Роутеры) разделенные по версиям/модулям
│   ├── core/           # Конфигурация (pydantic-settings), безопасность, логгер
│   ├── models/         # Модели базы данных (ORM)
│   ├── schemas/        # Схемы валидации Pydantic
│   ├── services/       # Бизнес-логика (Service Layer)
│   └── main.py         # Точка входа в приложение
│
├── docker-compose.yml  # Инфраструктура (App, DB, Redis)
├── Dockerfile          # Сборка приложения
├── requirements.txt    # Зависимости
└── README.md
```
# Запуск:
Создание контейнеров:

```sh
$ docker-compose up -d --build
```

Миграции:

```sh
$ docker-compose exec backend aerich upgrade
```
